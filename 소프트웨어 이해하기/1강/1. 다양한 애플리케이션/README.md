# 다양한 애플리케이션

## 세상에 존재하는 프로그램의 종류

>애플리케이션

```
프로그램을 만드는 사람이 아닌 실제 사용자 즉, 소비자가 직접 사용하는 프로그램을 애플리케이션이라 한다

사용자, 소비자 = End user

우리가 흔히 사용하는 스마트폰에서 다운로드 하는 것을 애플리케이션이라고 한다
하지만 컴퓨터에서도 마찬가지로 우리가 사용하는 프로그램을 애플리케이션이라 할 수 있다

컴퓨터: 카카오톡, 스와이프, 크롬, 사파리, 파워포인트, 엑셀 등

스마트폰: 카카오맵, 지도, 카카오톡, 틱톡, 은행어플, 사진어플 등

에어컨, 선풍기, TV 등도 리모컨 혹은 버튼으로 조작을 가하고 동작시킬 수 있기 때문에 일종의 간단한 애플리케이션이다
```

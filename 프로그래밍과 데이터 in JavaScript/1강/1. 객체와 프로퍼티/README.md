# 객체와 프로퍼티

## 객체 (object)

>여러가지 값을 저장하고 싶을 때 사용하는 것

객체는 JavaScript의 거의 모든 문범에 녹아있는 개념이다

>객체 만드는 법

```
{
  brandName: '이현석',
  bornYear: 2022,
  isVeryNice: true,
  worstCourse: null
}

왼쪽은 Key 오른쪽은 Value
Key - Value 한 쌍을 속성 (property)라고 불른다
```

>Property Name 주의 사항

1. 첫번째 글자는 반드시 **문자, 밑줄(_), 달러 기호($) 중 하나로 시작**

2. 띄어쓰기 금지

3. 하이픈 (-) 금지

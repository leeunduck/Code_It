# 다양한 링크

>기본 글자 링크 (inline 요소)
```
<a href="주소">글자 내용</a>
```

>이미지를 사용한 링크 (inline 요소)
```
<a href="주소">
  <img src="이미지 파일 이름.png>
</a>
```

>여러 요소를 합친 링크
```
html

<a class="address" href="주소">
  <img src="이미지 파일 이름.png>
  <h1>제목</h1>
  <p>내용</p>
</a>
```

```
css

text-align: center;
padding: 20px;
border-radius: 10px;
border: 1px solid #eee;
width: 300px;
background-color: #fafafa;
margin-top: 100px

display: block;
color: black;
text-decoration: none;
```

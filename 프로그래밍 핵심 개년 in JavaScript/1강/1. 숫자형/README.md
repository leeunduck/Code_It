# 숫자형

>사칙연산

```
// 덧셈
console.log(1 + 8);

// 뺄셈
console.log(6 - 7);

// 곱셈
console.log((2 * 2);

// 나눗셈
console.log(5 / 2);

출력 값: 9, -1, 4, 2.5
```

>그 외 연산

```
// 나머지
console.log(7 % 3);
7을 3으로 나눈 나머지를 구하는 것

// 거듭제곱
console.log(2 ** 3);
2의 3제곱

// 연산 우선순위
console.log(2 + 3 * 2);
곱셈이 먼저 계산 됨

// 연산 우선순위 2
console.log((2 + 3) * 2);
```

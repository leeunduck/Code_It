# 나의 첫 번째 웹사이트

```
<!DOCTYPE html> ➡️ 웹 브라우저에게 HTML 버전을 알려주는 역할 (이렇게 쓰면 자동으로 HTML5 사용)
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title> ➡️ 웹사이트의 제목
</head>
<body>
    <h1>My First Page</h1> ➡️ 가장 큰 머리말 (heading 1)
</body>
</html>
```

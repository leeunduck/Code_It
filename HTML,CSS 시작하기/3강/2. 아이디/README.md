## 클래스와 아이디의 차이

>class는 여러 요소에 같은 스타일을 입힐 때 사용할 수 있다 (한 태그에 여러 중복 클래스를 쓸 수 있다)
>
>ID는 중복이 불가능하기 때문에 각 페이지에서 중복되는 ID가 있으면 안된다 (한 태그에 여러 중복 클래스를 쓸 수 없다)

여러 요소를 스타일링 하고 싶으면? ➡️ Class

한 요소만 스타일링 하고 싶으면? ➡️ ID
